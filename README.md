# zhcn-cn-journal

本仓库用于归档和发布多个独立 HTML 页面。这些页面以静态形式存储，便于查阅和分享。仓库内容不针对任何单个域名或具体网站，仅作为页面归档的集合。

## 项目简介

zhcn-cn-journal 是一个面向通用 HTML 页面存档的仓库。所有页面以独立文件形式存在，可直接通过浏览器打开预览。仓库维护者定期收录、整理并发布这些页面，供有需要的人参考或使用。

## 目录说明

```
.
├── index.html          # 仓库入口页面（可选）
├── pages/              # 存放归档的 HTML 页面
│   ├── example.html
│   └── ...
├── assets/             # 静态资源（图片、样式、脚本等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：所有归档的 HTML 页面存放于此。
- `assets/`：页面所需的公共资源文件。
- 根目录下的 `index.html` 可作为导航首页（如存在）。

## 页面归档说明

- 归档的页面均为独立 HTML 文件，可离线打开。
- 页面内容保持原始呈现，不添加额外修改。
- 归档时间、来源等信息可在页面内或文件名中体现（如有）。
- 仓库不保证所有页面的持续可用性，亦不承担页面内容的法律责任。

## 维护说明

- 本仓库由维护者不定期更新。
- 新增页面时，请将 HTML 文件放入 `pages/` 目录，并确保相关资源引用正确。
- 如需提交页面归档，欢迎通过 Pull Request 或 Issue 联系。
- 仓库维护者保留对收录内容的筛选与删除权利。
